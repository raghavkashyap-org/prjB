Machine Learning Modules

Supervised Learning

Focused on detecting known attack categories using Random Forest, XGBoost, SVM, and Logistic Regression.

Attacks: DoS/DDoS, Brute Force, Malware, Phishing, Port Scan

Unsupervised Learning

Designed for unknown attack detection using Isolation Forest, K-Means, and DBSCAN for density-based anomalies.

Deep Learning Engine

Autoencoder

Detects anomalies by monitoring reconstruction errors in network behavior sequences.

LSTM (Time-Series)

Focuses on temporal behavior patterns to predict future attack probability.

Autoencoder Reconstruction Loss
$$Loss = ||X - X'||$$

Large error \(\rightarrow\) Anomaly detected

Production & Deployment

Real-Time Stack

Scapy (Capture), FastAPI/Flask (Backend API), and Streamlit (Live Dashboard).

Containerization

Docker is used to containerize the Model, API, and Dashboard for portable deployment.

Cloud Hosting

Production simulation on AWS EC2 or Azure VM to mimic enterprise security environments.

Secure Log Integrity

Hash Chain

$$Hash_{n} = SHA256(Log_{n} + Hash_{n-1})$$

Ensures that tampering breaks the chain of logs.

Blockchain (Ethereum)

Stores alert hashes in smart contracts to ensure logs are impossible to modify.
